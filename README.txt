== Description ==

The Tilt Cryptocurrency Payments plugin allows you to accept cryptocurrency payments from your WooCommerce store directly into a private wallet. This plugin connects to the Tilt API in order to generate addresses, determine exchange rates and to monitor when payments are completed.

Please visit the [Tilt website](https://tilt.cash) for more details about configuring and running your own Tilt wallet instance.

Features:

*   Accept payments directly into your private wallet.
*   No approval process / no account required.
*   No middleman transaction fees.
*   Your Keys, Your Coins.
*   Support for Bitcoin, Litecoin, Dogecoin and more soon.
